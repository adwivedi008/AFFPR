Rules for HTML

1. Use no space in page name and title.
2. Head section only contain Tab name.
3. Use space in between properties.
4. div can have heading(h1,h2,etc.)but heading(h1,h2,etc.)doen't have div.
5. For nesting of tags ,use div instead of p tag.
6. Always terminate(;) style property.
(what is document class in browser is body section in HTMl)
